PUPC - Digitalized Document Storage System

Overview

A redesigned and modernized document management platform built for the Polytechnic University of the Philippines Calauan Campus. The system serves as a locally hosted alternative to cloud storage platforms, allowing faculty members to upload files, organize folders, and collaborate securely within the campus network. This revamp not only improved the user interface but also added the missing functionalities required by the faculty and campus administrators.

The platform emphasizes transparency and accountability by offering detailed activity logs, allowing users to track every change made to their files and folders. It now also includes a dedicated module for the campus registrar, enabling them to manage student records and upload entrance credentials and other important documents.

Tech Used: Laravel, Vue.js, Inertia.js, MySQL, Tailwind CSS

Key Features

  • File & Folder Management: Upload files, create folders, move, rename, and share items with ease.
  • Role-Based Access: Visibility and permissions change based on user roles, with options to share specific items across roles or with select individuals.
  • Activity Log: Tracks all updates, uploads, deletions, and changes for better accountability.
  • Registrar Module: Allows registrars to add students and upload their entry documents and requirements for secure tracking.
