Loan Management System

Overview

A modern web app built to simplify how La Famiglia Lending Investors Corporation handles loans, borrowers, and payments. It automates the entire process — from creating a loan to tracking repayments and generating reports — all in one clean, easy-to-use dashboard.

With this system, admins can manage borrower profiles, calculate amortization schedules instantly, and monitor real-time payment updates. It’s designed with scalability in mind, making it suitable for small local lenders or larger financial institutions that need a digital edge.

Tech Used: Laravel, Vue.js (Composition API), Inertia.js, MySQL, Axios, Chart.js, Tailwind CSS

Key Features

  • Borrower Management: Add, update, and monitor client profiles and loan histories.
  • Loan Processing: Generate loan schedules automatically based on interest rate, term, and payment frequency.
  • Payment Tracking: Record repayments and update remaining balances in real time.
  • Reports & Analytics: View summaries of active loans, paid-off accounts, and overdue balances.
  • User Roles & Permissions: Admin, staff, and auditor-level access for secure operations.
  • Dashboard: Visual analytics with charts for loan performance, collection rate, and profit margin.

The Story Behind It

This project was developed for my current company, La Famiglia Lending Investors Corp (LFLIC). We were using a third-party loan management software (LoanDisk) that charged a monthly subscription fee, which wasn’t sustainable long-term. To eliminate that recurring cost and gain full control over the system, I built an in-house solution tailored exactly to our processes. It was a rewarding experience — not just in reducing expenses, but in creating a system that truly fits how our team works.

What I'm Proud Of

Developing this project taught me a lot — from architecting complex loan logic to designing a smooth user experience. It deepened my understanding of how business rules translate into code and how thoughtful design can make financial workflows faster and more reliable.
