My Projects

Loan Management System

A full-featured web app for managing loans, borrowers, payments, and amortization schedules. Built for LFLIC to replace a subscription-based system, it streamlines operations with a modern dashboard, automated calculations, and real-time tracking. It’s designed to be scalable, efficient, and tailored to real lending workflows.

Point of Sales (POS) System

A restaurant-focused POS built for UPTOP Diner, featuring real-time order updates that flow directly to the kitchen. It handles sales, menu management, and revenue insights in a fast, intuitive interface designed for busy service environments. The system improves accuracy and speed across both front-of-house and kitchen operations.

Basic Inventory System

A lightweight inventory tool built with Laravel, Vue, and Inertia.js for simple item management. It supports adding items, recording sales, importing/exporting data, and generating quick daily to monthly reports. Perfect for small businesses needing a straightforward, easy-to-maintain inventory solution.

PUP Calauan — Digitalized Document Storage System

A redesigned, locally hosted document management platform for PUP Calauan that lets faculty upload files, organize folders, and track changes through detailed activity logs. The revamp improved the interface, added missing features, and introduced a registrar module for managing student records and entrance credentials. It delivers a secure, campus-focused alternative to cloud storage with role-based access and collaboration.
